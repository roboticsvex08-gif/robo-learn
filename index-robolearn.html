🛠️ Quick Setup

5-Minute Setup Guide

1

Install Node.js & MongoDB

Download Node.js from nodejs.org and MongoDB from mongodb.com

2

Install Backend Dependencies

Navigate to backend folder and run:

cd robolearn-platform/backend && npm install
3

Configure Environment

Create .env file in backend folder with:

PORT=5000
MONGODB_URI=mongodb://localhost:27017/robolearn
JWT_SECRET=your_secret_key
NODE_ENV=development
4

Start Backend Server

From backend folder, run:

npm start
5

Open Frontend

Open in browser:

robolearn-platform/frontend/public/index.html

First Time Usage

  1. Click "Admin Panel" or go to robolearn-platform/frontend/admin/login.html
  2. Click "Create one here" to register a new admin account
  3. Fill in username, email, and password
  4. After registration, go to "Upload Project"
  5. Fill in all project details and upload
  6. Your project will instantly appear on the public showcase page!
